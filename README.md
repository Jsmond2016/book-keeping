# React-Typescript 记账项目

**主要技术栈**

- React，React-Router
- Typescript
- dayjs 等

## 功能说明

- 标签页：新增、更改标签
- 记账页：记录支出/收入 账款，并匹配标签
- 统计页：账单查看，历史 支出/收入

## 项目启动

- 下载本项目

```bash
git clone https://github.com/Jsmond2016/book-keeping.git
```

- 项目启动

```bash
yarn start
```

- 预览地址

```bash
http://localhost:3000
```
